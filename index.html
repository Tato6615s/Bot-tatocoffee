วิธีเชื่อมต่อ Backend

เว็บนี้ปัจจุบันเก็บข้อมูลใน localStorage (ชั่วคราว) — เพื่อใช้งานจริงต้องเชื่อมต่อ Server

ง่ายที่สุด 🔥 Firebase / Supabase (No-code Backend)

เหมาะกับร้านขนาดเล็ก-กลาง ไม่ต้องมี Server เอง ฟรีได้ถึง 50K request/เดือน

1. สมัคร firebase.google.com หรือ supabase.com
2. สร้าง Database → collections: coffees, orders, reviews
3. แทนที่ฟังก์ชัน saveData() และ loadData() ด้วย SDK
4. ตั้ง Security Rules ให้ Admin เท่านั้นเขียนได้
📄 Firebase Docs → 📄 Supabase Docs →
แนะนำ 🌐 Host บน Netlify / Vercel (ฟรี)

อัปโหลดไฟล์ index.html ขึ้น host แล้วได้ URL ทันที รองรับ Custom Domain

1. ไปที่ netlify.com → "Add new site" → "Deploy manually"
2. ลาก index.html ไปวางในช่อง
3. ได้ URL ฟรี เช่น your-site.netlify.app
4. เชื่อม Custom Domain ของคุณเอง (เช่น origincoffee.th)
สำหรับออเดอร์จริง 📬 เชื่อม LINE Notify / Google Sheets

เมื่อลูกค้าสั่งซื้อ ให้ส่งข้อมูลเข้า Google Sheets + แจ้งเตือน LINE ทันที

1. สร้าง Google Apps Script → Web App URL
2. แทนที่ฟังก์ชัน submitOrder() ให้ POST ข้อมูลไป Script
3. Script เขียนลง Google Sheets + ส่ง LINE Notify
4. ได้รับ Notify ทุกครั้งที่มีออเดอร์ใหม่ทันที
🔐 ความปลอดภัย — สิ่งที่ต้องทำก่อน Go Live
✅ เปลี่ยนรหัสผ่าน Admin จาก origin2024 ทันที (ใน Admin → ตั้งค่า)
✅ ย้าย ADMIN_PASS ออกจาก JavaScript ไปเก็บบน Server (env variable)
✅ เปิด HTTPS เท่านั้น (Netlify/Vercel ทำให้อัตโนมัติ)
✅ ตั้ง Rate Limit บน API endpoint ป้องกัน Spam Order
✅ Validate ข้อมูลทั้ง Client และ Server ก่อนบันทึก
